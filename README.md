# FoodOrderingApp
This is a food ordering application which is client can buy foods online by ordering them.This app has an admin side also.The admin can add food items and their details to the food catalog.In here,if a client needs to add foods into the cart and buy them,he/she has to log as a client into the system. I have used MERN Stack to develop this application and used CSS/HTML to design the user interfaces.The IDE which I have used is Visual Studio.
